# Archive Directory

This directory contains archived specifications that are no longer active but kept for historical reference.

## Archived Specs

### feature-001-legacy
- **Archived**: 2025-10-24
- **Reason**: Legacy format, replaced by modern GitHub Spec Kit format specs (001-004)
- **Original Purpose**: Test spec for user login system
- **Status at Archive**: Draft

## Archive Policy

Specs are archived when:
1. They've been replaced by newer specifications
2. They're in legacy formats that have been migrated
3. They're no longer relevant to the project

## Restoration

To restore an archived spec:
```bash
mv .specify/_archive/spec-name .specify/specs/
```
