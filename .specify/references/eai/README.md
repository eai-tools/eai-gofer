# EnterpriseAI Local Reference Pack

This folder contains local fallback references used when external EnterpriseAI
repositories or documentation are unavailable at runtime.

## Included references

- `eai.md`
- `vertical-template.md`
- `deployment-repo.md`

## Usage contract

1. Prefer external sources when reachable.
2. Fall back to these local references when external sources are unavailable.
3. Record when fallback is used in generated artifacts.
