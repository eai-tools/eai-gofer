# Implementation Plan: Memory System v2 - Agent Memory Architecture

**Branch**: `029-memory-system-v2` | **Date**: 2026-03-19 | **Spec**:
[spec.md](./spec.md) | **Research**: [research.md](./research.md)

## Summary

Implement a comprehensive memory system that serves all Gofer agents with
measurable quality improvements. Extends existing ~3,700 LOC memory
infrastructure (MemoryManager, ContextBuilder, MemoryLayerManager) with three
critical enhancements: (1) **L0/L1/L2 tiered loading** (OpenViking-inspired) to
achieve 30-60% context token reduction, (2) **Sub-agent memory injection** for
validation/research agents to access past learnings (5-10 prioritized memories
per agent), and (3) **Automatic pattern extraction** from validation findings
and engineering reviews to close the learning loop.

**Primary Goals**: Achieve 95-100/100 validation scores (up from 85-95), reduce
context usage to <50k tokens by stage 5 (down from 100-150k), and decrease
repeated mistakes to <5% (down from ~20%).

**Technical Approach**: Adopt OpenViking's proven patterns (L0/L1/L2 layers,
gofer:// URI abstraction, directory recursive retrieval) implemented natively in
TypeScript. Enhance existing components incrementally rather than rewriting.
Prioritize sub-agent memory access (P0) and automatic extraction (P0) for
immediate quality gains.

```typescript
async loadByURI(uri: string, layer: 'L0' | 'L1' | 'L2' = 'L1'): Promise<Memory | null> {
  const resolved = this.uriResolver.resolve(this.uriResolver.parse(uri));
  const memory = await this.storage.loadByPath(resolved);

  if (!memory) return null;

  // Apply layer selection
  if (layer === 'L0' && memory.layers) {
    return { ...memory, content: memory.layers.abstract };
  } else if (layer === 'L1' && memory.layers) {
    return { ...memory, content: memory.layers.overview };
  } else {
    // L2 or no layers: return full content
    return memory;
  }
}
```

```typescript
// ContextBuilder.ts
async calculateCoverage(
  keywords: string[],
  memories: Memory[]
): Promise<number> {
  // Extract keywords from all memories
  const memoryKeywords = new Set<string>();
  for (const memory of memories) {
    const extracted = this.keywordExtractor.extract(memory.content, 20);
    extracted.forEach(kw => memoryKeywords.add(kw.toLowerCase()));
  }

  // Calculate coverage percentage
  const covered = keywords.filter(kw =>
    memoryKeywords.has(kw.toLowerCase())
  ).length;

  return keywords.length > 0 ? (covered / keywords.length) * 100 : 0;
}

async buildContextWithCoverageOptimization(
  taskContext: string,
  researchDocs: string[]
): Promise<string> {
  const sections: string[] = [];

  // 1. Load memories from all scopes
  const memories = await this.memoryManager.loadByPriority({
    taskContext,
    limit: 20,
    scope: 'all',
  });

  // 2. Extract keywords from task context
  const keywords = this.keywordExtractor.extract(taskContext, 30);

  // 3. Calculate coverage
  const coverage = await this.calculateCoverage(
    keywords.map(k => k.word),
    memories
  );

  this.emit('coverage-calculated', { coverage, threshold: 30 });

  // 4. Decision: skip research docs if coverage >= 30%
  if (coverage >= 30) {
    this.emit('loading-decision', {
      source: 'research-docs',
      decision: 'skipped',
      reason: `Coverage ${coverage.toFixed(1)}% >= 30% threshold`,
      tokens: 0,
    });

    // Only load memories, skip research docs
    sections.push(await this.formatMemories(memories, 40000));
  } else {
    this.emit('loading-decision', {
      source: 'research-docs',
      decision: 'loaded',
      reason: `Coverage ${coverage.toFixed(1)}% < 30% threshold`,
      tokens: Math.ceil(researchDocs.join('').length / 4),
    });

    // Load both memories and research docs
    sections.push(await this.formatMemories(memories, 10000));
    sections.push(researchDocs.join('\n\n'));
  }

  return sections.join('\n\n');
}
```

```typescript
  }
}

// package.json contribution
"gofer.memory.coverageThreshold": {
  "type": "number",
  "default": 30,
  "minimum": 0,
  "maximum": 100,
  "description": "Keyword coverage percentage threshold for skipping research docs (FR-004). If memory coverage >= threshold, research docs are skipped."
}

export class SubAgentContextFactory {
  private coverageThreshold: number;

  constructor(
    private memoryManager: MemoryManager,
    private contextBuilder: ContextBuilder,
    private uriResolver: GoferURIResolver,
    private configManager: ConfigManager
  ) {
    this.coverageThreshold = configManager.getCoverageThreshold();
  }

  async buildValidationContext(
    category:
      | 'correctness'
      | 'security'
      | 'performance'
      | 'integration'
      | 'test-quality'
      | 'standards',
    featureDir: string,
    tokenBudget: number = 8000
  ): Promise<string> {
    const sections: string[] = [];

    // 1. Load prioritized memories for this category FIRST
    const memories = await this.memoryManager.loadByPriority({
      taskContext: `Validate ${category} for feature`,
      limit: 10,
      scope: 'local',
      tags: [`#${category}`, '#validation'],
    });

    // 2. Calculate keyword coverage (FR-004)
    const keywords = this.contextBuilder.keywordExtractor.extract(
      `Validate ${category} for feature in ${featureDir}`,
      30
    );
    const coverage = await this.contextBuilder.calculateCoverage(
      keywords.map(k => k.word),
      memories
    );

    // 3. Decision: skip spec docs if coverage >= threshold (FR-004)
    if (coverage >= this.coverageThreshold) {
      // HIGH COVERAGE: Skip spec overview, use memories only
      this.contextBuilder.emit('loading-decision', {
        source: `spec-${featureDir}`,
        decision: 'skipped',
        reason: `Coverage ${coverage.toFixed(1)}% >= ${this.coverageThreshold}% threshold`,
        tokens: 0,
        layer: 'L1',
      });

      // Use full token budget for memories
      const memoryContext = await this.formatMemories(memories, tokenBudget);
      sections.push(`## Past Validation Patterns (${coverage.toFixed(1)}% coverage)\n\n${memoryContext}`);
    } else {
      // LOW COVERAGE: Load both spec and memories
      this.contextBuilder.emit('loading-decision', {
        source: `spec-${featureDir}`,
        decision: 'loaded',
        reason: `Coverage ${coverage.toFixed(1)}% < ${this.coverageThreshold}% threshold`,
        tokens: 2000,
        layer: 'L1',
      });

      const specURI = `gofer://specs/${featureDir}/spec.md`;
      const specOverview = await this.loadLayerWithBudget(specURI, 'L1', 2000);
      sections.push(`## Spec Overview\n\n${specOverview}`);

      const memoryContext = await this.formatMemories(
        memories,
        tokenBudget - 2000
      );
      sections.push(`## Past Validation Patterns (${coverage.toFixed(1)}% coverage)\n\n${memoryContext}`);
    }

    // 4. Add validation criteria for this category (always included)
    const criteria = this.getValidationCriteria(category);
    sections.push(`## Validation Criteria\n\n${criteria}`);

    return sections.join('\n\n---\n\n');
  }

  private async formatMemories(
    memories: Memory[],

    for (const memory of memories) {

      const tokens = Math.ceil(content.length / 4);

      if (tokensUsed + tokens > tokenBudget) break;

      formatted.push(
        `### ${memory.category} - ${memory.tags.join(', ')}\n\n` +
          `${content}\n\n` +
          `**Citations**: ${memory.citations?.map((c) => `${c.file}:${c.line}`).join(', ') || 'None'}\n` +
          `**Used**: ${memory.usedCount} times, Last: ${memory.lastUsed}`
      );

      tokensUsed += tokens;
    }
```

```typescript
  constructor(private logger: Logger) {}

  /**
   * Track which memories were injected into a sub-agent context

  trackInjectedMemories(
    agentType: string,
    agentCategory: string,
    memoryIds: string[]
  ): void {

      agentType,
      agentCategory,
      memoryIds,
      timestamp: Date.now(),

  /**
   * Verify that validation report cites the injected memories
   * Returns: { cited: string[], missing: string[], citationRate: number }
   */
  async verifyMemoryCitations(
    reportPath: string,
    injectedMemoryIds: string[]
  ): Promise<{
    cited: string[];
    missing: string[];
    citationRate: number;
  }> {
    const reportContent = await fs.readFile(reportPath, 'utf-8');

    const cited: string[] = [];
    const missing: string[] = [];

    for (const memoryId of injectedMemoryIds) {
      // Check if memory ID or its content appears in the report
      // Memory citations format: "Per memory mem-001234..." or "[mem-001234]"
      const patterns = [
        `mem-${memoryId}`,
        `memory ${memoryId}`,
        `[${memoryId}]`,
      ];

      const wasCited = patterns.some(pattern =>
        reportContent.toLowerCase().includes(pattern.toLowerCase())
      );

      if (wasCited) {
        cited.push(memoryId);
      } else {
        missing.push(memoryId);
      }
    }

    const citationRate =
      injectedMemoryIds.length > 0
        ? (cited.length / injectedMemoryIds.length) * 100
        : 0;

    return { cited, missing, citationRate };
  }

  /**
   * Log citation metrics to observability system

  logCitationMetrics(
    agentCategory: string,
    cited: string[],
    missing: string[],
    citationRate: number
  ): void {
    this.logger.info('Memory citation metrics', {
      agentCategory,
      citedCount: cited.length,
      missingCount: missing.length,
      citationRate: `${citationRate.toFixed(1)}%`,
      status: citationRate >= 50 ? 'PASS' : 'WARNING',

  }
```

```typescript
    (e) => e.eventType === 'loading_decision'
  );

  // Group by stage
  const byStage = new Map<string, LoadingDecision[]>();
  for (const event of loadingEvents) {
    const stage = event.stage || 'unknown';
    if (!byStage.has(stage)) byStage.set(stage, []);
    byStage.get(stage)!.push(event);
  }

  // Report per stage
  for (const [stage, decisions] of byStage) {
    const loaded = decisions.filter((d) => d.decision === 'loaded');
    const skipped = decisions.filter((d) => d.decision === 'skipped');
    const totalTokens = loaded.reduce((sum, d) => sum + (d.tokens || 0), 0);

    outputChannel.appendLine(`## Stage: ${stage}`);
    outputChannel.appendLine(
      `- Loaded: ${loaded.length} memories (${totalTokens} tokens)`
    );
    outputChannel.appendLine(`- Skipped: ${skipped.length} memories`);
    outputChannel.appendLine('');
  }

  // Most loaded memories
  const memoryLoadCounts = new Map<string, number>();
  for (const event of loadingEvents) {
    if (event.decision === 'loaded') {
      memoryLoadCounts.set(
        event.source,
        (memoryLoadCounts.get(event.source) || 0) + 1
      );
    }
  }

  const topMemories = Array.from(memoryLoadCounts.entries())
    .sort((a, b) => b[1] - a[1])
    .slice(0, 10);

  outputChannel.appendLine('## Most Loaded Memories');
  for (const [memoryId, count] of topMemories) {
    outputChannel.appendLine(`- ${memoryId}: ${count} times`);
  }
}
```

```typescript
export class MemoryStorage {
  private l2Cache = new Map<string, string>(); // LRU cache (max 50 entries)
  private cacheMaxSize = 50;

  async loadDetail(memoryId: string): Promise<string> {
    // Check cache first
    if (this.l2Cache.has(memoryId)) {
      return this.l2Cache.get(memoryId)!;
    }

    // Load from file
    const memory = await this.loadById(memoryId);
    const detail =
      memory.content.length > 500
        ? await fs.readFile(
            `.specify/memory/memory-notes/${memoryId}.md`,
            'utf-8'
          )
        : memory.content;

    // Cache with LRU eviction
    if (this.l2Cache.size >= this.cacheMaxSize) {
      const firstKey = this.l2Cache.keys().next().value;
      this.l2Cache.delete(firstKey);
    }
    this.l2Cache.set(memoryId, detail);

    return detail;
  }
}
```

````typescript
export class TokenEstimator {
  // Simple heuristic-based estimator (no external tokenizer dependency)
  estimate(text: string): number {
    // Baseline: 4 chars per token
    let tokens = Math.ceil(text.length / 4);

    // Adjust for code (more tokens per char)
    const codeBlocks = text.match(/```[\s\S]*?```/g) || [];
    const codeChars = codeBlocks.reduce((sum, block) => sum + block.length, 0);
    tokens += Math.ceil(codeChars / 3) - Math.ceil(codeChars / 4); // +33% for code

    // Adjust for markdown (fewer tokens per char)
    const headings = text.match(/^#+\s/gm) || [];
    tokens -= headings.length * 2; // Headings compress well

    return tokens;
  }

  estimateBatch(texts: string[]): number {
    return texts.reduce((sum, text) => sum + this.estimate(text), 0);
  }
}
````

## Key Architecture Decisions

### Decision 1: Adopt OpenViking Patterns, Not Full Integration

**Rationale**: OpenViking's L0/L1/L2 tiered loading and URI abstraction are
proven (91-96% token reduction), but full Python integration adds complexity.
Implementing patterns natively in TypeScript preserves simplicity while
achieving similar benefits.

**Impact**:

- Maintains TypeScript-only ecosystem (no Python bridges)
- Achieves 30-60% token reduction (vs. 91-96% with full OpenViking)
- Keeps git-friendly JSONL storage
- Enables incremental adoption

**Alternative Rejected**: Full OpenViking integration - requires Python,
embedding models, complex setup.

---

### Decision 2: Extend Existing Components Instead of Rewriting

**Rationale**: Gofer already has ~3,700 LOC of memory infrastructure
(MemoryManager, ContextBuilder, MemoryLayerManager) that works well. Rewriting
risks introducing bugs and breaking existing functionality.

**Impact**:

- Minimizes risk (smaller diffs, focused changes)
- Preserves institutional knowledge in existing code
- Maintains backward compatibility
- Faster implementation (3-4 weeks vs. 8-12 for rewrite)

**Alternative Rejected**: Clean-slate rewrite - too disruptive, higher risk.

---

### Decision 3: Prioritize Sub-Agent Memory Access (P0) Over Architecture Migration

**Rationale**: Research shows sub-agent memory injection (5-10 memories per
agent) directly impacts quality (10-15% validation score improvement). Sub-agent
dispatch architecture (fresh 200k context per stage) is a larger change deferred
to future work.

**Impact**:

- Immediate quality gains (P0 user stories US-P1-01, US-P1-02)
- Smaller scope reduces risk
- Context bloat still reduced 30-60% via tiered loading
- Sub-agent dispatch can be added later (decoupled)

**Alternative Rejected**: Tackle sub-agent dispatch first - larger scope, delays
quality improvements.

---

### Decision 4: LLM-Based Layer Generation with Graceful Degradation

**Rationale**: Claude Haiku can generate L0/L1 layers cost-effectively (~$0.001
per memory), but failures shouldn't block extension. Non-blocking design with
fallbacks ensures robustness.

**Impact**:

- Auto-generation at save time (users don't manually write summaries)
- Haiku model keeps costs low (~$1-2/month)
- Failures logged but don't crash extension
- Backward compatibility: old memories load via L2 fallback

**Alternative Rejected**: Rule-based summarization - insufficient quality,
misses nuances.

---

### Decision 5: Backward Compatibility as Hard Requirement

**Rationale**: Users have existing memory collections (~100-500 memories per
project). Breaking changes would require manual migration or data loss, harming
user trust.

**Impact**:

- All existing memories load without modification
- Migration opt-in and non-destructive
- API surface preserved (additive only)
- Users can continue using old format indefinitely

**Alternative Rejected**: Breaking changes with forced migration - high user
friction, risk of data loss.

---

### Decision 6: Observable Memory Loading for Debuggability

**Rationale**: Memory retrieval is non-deterministic (fuzzy keyword matching).
Observable decisions (logged to context-usage.jsonl) enable debugging "why
didn't the agent see this context?" and iterative tuning.

**Impact**:

- All loading decisions logged with rationale
- Memory panel UI shows last loaded info
- Query command provides stage-by-stage analysis
- Transparent system behavior

**Alternative Rejected**: Opaque retrieval - harder to debug, users can't
understand system behavior.
