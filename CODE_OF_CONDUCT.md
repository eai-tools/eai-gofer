# Code of Conduct

Gofer aims to be a friendly, practical, contributor-friendly project.

## Our Standard

We expect participants to:

- be respectful and constructive
- assume good intent while giving direct technical feedback
- help keep issues, pull requests, and discussions useful for everyone
- avoid harassment, hostility, and personal attacks

## Unacceptable Behavior

- abusive or discriminatory language
- harassment, intimidation, or dogpiling
- publishing private information without permission
- derailing support threads or contribution discussions in bad faith

## Reporting

If you experience or observe unacceptable behavior, contact the repository
maintainers through GitHub moderation and reporting channels instead of starting
a public argument.

This repository also aligns with the broader
[GitHub Community Code of Conduct](https://docs.github.com/en/site-policy/github-terms/github-community-code-of-conduct).
