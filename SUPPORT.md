# Support

## Where to Ask What

- **Questions and usage help**:
  [GitHub Discussions](https://github.com/eai-tools/eai-gofer/discussions)
- **Longer-form project reference**:
  [GitHub Wiki](https://github.com/eai-tools/eai-gofer/wiki)
- **Bug reports and regressions**:
  [GitHub Issues](https://github.com/eai-tools/eai-gofer/issues)
- **Security reports**: see [SECURITY.md](./SECURITY.md)

## Before Opening A Ticket

- Confirm which surface you are using: VS Code, Claude Code, Codex, GitHub
  Copilot, or Gemini.
- Include the command or stage you ran.
- Include the Gofer version and host version.
- Include the smallest reproducible repo or log excerpt you can safely share.
- If your issue matches the current public roadmap, Gofer may open a draft
  intake PR automatically to help a maintainer review it faster.
