# Archive

This folder holds material that no longer reflects the current canonical EAI
Gofer repo state but is being retained for historical reference.

## 2026-05 Repo Audit

`2026-05-repo-audit/` contains files moved out of the active repo surface during
the May 29, 2026 audit. These items fell into one or more of these buckets:

- superseded root docs that were no longer referenced by the current workflow
- obsolete helper scripts replaced by canonical `.specify` or `release.sh`
- tracked runtime or backup artifacts that should not remain in the public repo
- generated dashboard output that can be recreated on demand

Nothing in this folder is treated as a source-of-truth input for builds, release
packaging, or the active EAI Gofer pipeline.
